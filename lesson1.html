Lesson 1: Prerequisites

Alright, before you start making your webpage, you need to grab a text editor. Mac users, Textedit WILL NOT work. I would recommend TextWrangler on the Mac App Store or Sublime Text at SublimeText.com. Windows users, you guys have a lot more choices, so I would just peek around the web for one that you like. I use Notepad++ from Notepad-Plus-Plus.org. Once you have that downloaded and/or installed, go ahead and open it. The following pictures accompaning this tutorial will be in Notepad++, but its about the same for every editor.

Once the program has opened, you should be presented with a blank document. If you are presented with a screen that has other buttons like "Open" or "New", click "New". If you need more assistance starting your editor, consult the help pages online, or in the program. Go ahead and add "" to the top line of the document. This is going to tell the computer that this is a html document. Go to line 2, and type in "". This tells the computer that this is the start of the HTML code. Press enter to go down another line, and press tab. This should go forward on the line. Then, type "". This is the start of the "head" element. The cursor should go straight down, not back to the beginning of the line. If it does, simply press tab again to get back where you where. Then press tab AGAIN to go forward again.
